Windows 1.0 wasn't quite the powerhouse it is today. It was basically a graphical skin for MS-DOS that allowed you to use a mouse and run several programs at once. 

It was famously delayed for years, and when it finally arrived, critics weren't sure it would catch on. But it laid the foundation for the most dominant operating system in history. It gave us our first taste of the Start menu (eventually) and the blue screen of death (unfortunately). 🖼️
