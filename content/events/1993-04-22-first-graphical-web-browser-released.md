Before Mosaic, the web was mostly text. Marc Andreessen and Eric Bina at the NCSA created a browser that could display images alongside text on the same page. 

It was the 'spark' that ignited the web explosion. Suddenly, the internet wasn't just for scientists; it was for everyone. Mosaic led to Netscape, which led to the browser wars and the internet we know today. It turned the web into a visual, interactive medium. 🌐
